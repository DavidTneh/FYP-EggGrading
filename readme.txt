Right click this folder and choose open with visual studio code

Laravel requirements:
1. php version: 8.2.12
2. install composer: https://getcomposer.org/download/
3. Run the commands at terminal:
composer install
composer update
4. php artisan key:generate
5. php artisan app:run-migrations
6. php artisan storage:link
7. php artisan serve --port=8001
